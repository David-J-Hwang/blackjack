# Blackjack

## 1. 개요
  - 블랙잭을 구현한 Javascript 프로젝트다.
  - 플레이어는 $1000을 가진 상태로 시작하고, $100, $500, $1000, $10000을 걸 수 있다. 플레이어는 소지금액이 베팅할 금액과 같거나 더 큰 경우에만 해당 선택지를 선택할 수 있다.
  - 트럼프카드는 일반적인 13*4 카드 세트로 시작하되, 추후 다른 조합의 카드 세트를 사용할 수 있다.
  - 플레이어와 딜러(컴퓨터)는 각각 카드 2장을 받고, 플레이어의 카드들은 모두 공개, 딜러의 카드는 한 장만 공개한다.
  - 처음 받은 2장이 (A, 10)이면 블랙잭이다. 블랙잭은 일반 21보다 더 강하지만, 딜러도 블랙잭이면 무승부가 된다.
  - 플레이어는 다음과 같은 의사결정을 할 수 있다.
    (1) Hit: 카드를 한 장 더 받는다.
    (2) Stand: 더 받지 않고 턴을 종료한다.
    (3) Double Down: 베팅한 금액을 두 배로 올리고 카드 한 장만을 더 받은 뒤 턴을 종료한다. 단 소지금액이 부족한 경우 실행할 수 없다.
    (4) (구현하지 않음)Split: 처음 두 장이 같은 값인 경우 손패를 두 개로 나눠서 플레이한다.
    (5) Surrender: 첫 의사결정에서 베팅한 금액의 절반을 되돌려받고 포기한다.
  - 플레이어의 점수가 21이 넘으면 즉시 패배한다.(Bust)
  - 플레이어가 Bust하지 않고 턴을 종료한 경우, 딜러는 숨긴 카드를 공개하고 카드를 뽑는다. 16까지는 반드시 Hit, 17부터는 Stand한다.
  - Soft17(A, 6)인 경우 이지모드에서는 딜러가 Stand하도록, 하드모드에서는 딜러가 Hit한다.
  - 딜러가 Bust하면 플레이어가 승리하고, 플레이어와 딜러가 모두 21 이하면 점수를 비교해 승패를 결정한다. 점수가 같으면 무승부(Push) 처리한다.
  - 플레이어가 승리한 경우 베팅한 금액의 2배를, 블랙잭으로 승리한 경우 베팅한 금액의 2.5배를 지급한다. 무승부의 경우 베팅한 금액을 반환하고, Surrender한 경우 베팅한 금액의 0.5배를 반환한다.
